# 🚀 AI Interview Agent  
### Real-Time HR & Technical Mock Interview Platform (MERN + AI)

An AI-powered full-stack platform that simulates real interview environments with **HR and technical modes**, enabling users to practice, get feedback, and improve—all in one place.

---

## 🧠 Overview

AI Interview Agent allows users to take **mock interviews anytime** using an AI interviewer.  
Users can select a role, add skills, or upload a resume, and the system generates relevant questions, evaluates answers, and provides feedback.

---

## ✨ Features

### 🎯 Smart Interview Setup
- Role & skills-based configuration  
- Resume upload with automatic skill & role detection  

### 🧩 Interview Modes
- **HR Mode** – behavioral & communication questions  
- **Technical Mode** – domain-specific / DSA questions  

### 🤖 AI Interview Experience
- Video-based AI interviewer  
- 5-question structured interview  
- Timer for each question  

### 🎤 Answer Methods
- Voice input (speech-to-text)  
- Text-based responses  

### 📊 Feedback System
- Instant AI-generated feedback  
- Strengths & improvement suggestions  
- Downloadable report  

### 💰 Token System
- New users get **100 tokens**  
- Each interview costs **50 tokens**  
- Online payments integrated  

### 📁 Dashboard
- View interview history  
- Track past performance  

### 🔐 Authentication
- Secure user login system

## ⚙️ Installation & Setup

### 1. Clone Repository
```bash
git clone https://github.com/your-username/ai-interview-agent.git
cd ai-interview-agent
```

### 2. Backend Setup
```bash
cd backend
npm install
```

Create `.env` file in backend:
```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
OPENAI_API_KEY=your_api_key
RAZORPAY_KEY_ID=your_key
RAZORPAY_SECRET=your_secret
FIREBASE_CONFIG=your_config
```

Run backend:
```bash
npm run dev
```

---

### 3. Frontend Setup
```bash
cd frontend
npm install
npm start
```
